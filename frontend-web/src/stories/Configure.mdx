# Bienvenido al Catálogo de Componentes de La Croassantina

Este es el entorno de Storybook para **La Croassantina**, nuestra pastelería y atelier de delicias diarias. Aquí podrás explorar, probar e interactuar con todos los componentes visuales e interactivos de la aplicación de forma aislada.

## Estructura del Proyecto

Los componentes están organizados de acuerdo con nuestro sistema de diseño:

- **Ejemplos**: Componentes básicos y de plantilla por defecto.
- **Layout**: Elementos estructurales como la cabecera (`Header`), el pie de página (`Footer`) y el avatar del usuario (`UserAvatar`).
- **Cards**: Tarjetas para mostrar productos en formato horizontal, vertical o de carga (`CardSkeleton`).
- **Cart**: Componentes de gestión del carrito (`CartItem` y `CartSummary`).
- **Product**: Paneles específicos del producto como el selector de cantidad y compra (`ProductActionCard`).
- **Purchase**: Paneles de información y líneas de detalle de pedidos (`PurchaseInfo`, `PurchaseLines`).
- **Sections**: Estructuras de contenido y listados (`ProductList`, `ProductSection`, `SectionBase`).
- **UI**: Elementos de control de interfaz de usuario como la paginación (`Pagination`).

## Cómo usar este Storybook

1. Selecciona cualquier componente de la barra lateral para ver su documentación y renderizado interactivo.
2. Usa el panel de **Controles** (Controls) para modificar las propiedades (props) en tiempo real y observar los cambios.
3. Consulta la pestaña **Actions** para comprobar el comportamiento de los eventos y callbacks.
